# Python Console Projects 🐍

A collection of five standalone Python console-based applications built using Python. These projects focus on core programming concepts such as file handling, object-oriented programming, exception handling, data structures, and modular programming while solving real-world problems.

---

## 📌 Projects Included

### 1. Expense Tracker 💰
A console application that helps users manage their finances by recording income and expenses, categorizing transactions, and generating reports.

**Features**
- Add income and expenses
- Categorize transactions
- View transaction history
- Generate monthly reports
- Calculate savings and balance

---

### 2. Habit Tracker 📅
A command-line application designed to help users build and maintain productive habits through daily tracking and progress statistics.

**Features**
- Create habits
- Mark daily completion
- Calculate streaks
- Generate weekly reports
- View progress statistics

---

### 3. Personal Password Manager 🔐
A secure console application for storing and managing credentials with password generation and strength checking capabilities.

**Features**
- Master password authentication
- Generate strong passwords
- Store credentials
- Search and update passwords
- Password strength checker

---

### 4. Banking Management System 🏦
A banking application that simulates fundamental banking operations and account management.

**Features**
- Create accounts
- Deposit money
- Withdraw money
- Transfer funds
- Transaction history
- Update account information
- Delete accounts

---

### 5. Quiz Application ❓
An interactive quiz system that allows users to attempt quizzes from multiple categories with score tracking and high-score management.

**Features**
- Multiple quiz categories
- Randomized questions
- Score calculation
- High-score management
- Replay option
